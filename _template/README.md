# ESDoc Template Plugin
## Install
```bash
npm install esdoc-template-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-template-plugin", "option": {"foo": true}}
  ]
}
```

- `foo` is default `true`
